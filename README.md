# Interpreter

Interpreter is a project that includes a custom programming language and an interpreter that translates and executes instructions of this language. The language supports both functional and object-oriented programming paradigms and is value-oriented rather than type-oriented.

## Features

- **Functional Programming:** Leverage functional programming constructs to write concise and expressive code.
- **Object-Oriented Programming:** Utilize object-oriented principles to structure and organize your programs.
- **Value-Oriented:** Focus on values rather than types, making the language flexible and dynamic.
